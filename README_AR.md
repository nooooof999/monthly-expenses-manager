# دليل استخدام البرنامج

## نظرة عامة
هذا البرنامج يساعد في إدارة المصروفات الشهرية بشكل فعال.

## المميزات الرئيسية
- واجهة مستخدم سهلة.
- تقارير مصروفات مخصصة.
- إمكانية إضافة مصروفات جديدة.

## طريقة الاستخدام
1. افتح البرنامج.
2. اختر "إضافة مصروف" وأدخل المعلومات المطلوبة.
3. اضغط على "حفظ" لتسجيل البيانات.

## أمثلة عملية
- إضافة مصروف لمشتريات الطعام.
- تتبع الفواتير الشهرية.

## حل المشاكل
- إذا لم يعمل البرنامج، تحقق من وجود جميع التحديثات.
